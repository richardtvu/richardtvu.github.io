The Life of Richard Vu at Cigna

Cigna is a health insurance company.

This is my first webpage written at Cigna. I started working at Cigna on May 13, 2019 and started development of this website on July 26, 2019. If you're interested in the process of how I developed this website, you can visit w3school.

My goal is to learn to use W3.CSS to make my own website and modify it to make it uniquely my own! I want to learn how to do these things:

  • Make a website from "scratch".
  • Make an input form for contacting me
  • Create a w3-ul class list
  • Make W3.CSS Tabs that will display information on Jordan Peterson, Nassim Taleb, and Neo. Specifically, I want to talk about how they've influenced me!
